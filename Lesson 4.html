2. Divs & Spans

Divs and Span are generic elements used to help style HTML

Div elements are block level elements. They allow you to group html element together. By default:

  • Div elements start on a new line
  • Elements after a div start on the next line
  • It takes up the full width of their parent elememt

Span elements are inline elements and are used to style individual sections of the elements. They make it easier to add little bits of formatting. By default:

  • Span elements do not start on a new line
  • Content after a span continues on the same line
  • It takes up only as wide as its content

3. Display

The CSS display property controls whether an element appears as an inline or a block level element. It can also control whether an element is rendered on the page at all

Display properties values are:
  • Inline
    • continue on the same line
    • only the width of their content
    • content after them will continue on the same line
    • height & width can't be controlled with CSS
  • Block
    • start on a new line
    • elements after them start on the next line
    • take up full width of their parent eleemt
    • height & width can be overridden using CSS height and weight properties
  • Inline-block
    • similar to inline element, except height and width can be controlled by CSS
  • None
    • it removes that element from the layout entirely and element is not rendered

Img tag. How about it? Inline or block?

  • It is an exception
  • It has a default display value of Inline, but acts as if it has the styling Inline-block
  • meaning its height and width can be adjusted but they continue on the same line as other element

4. IDs & Classes

IDs:
  • We can add an Id attribute to any HTML element to uniquely identify it
  • An element with a particular Id can be styled with CSS using # CSS selector
  • We can only use one Id to style one element
Classes:
  • We can add a Class attribute to identify similar elements on a page
  • An element with a particular class can be styled with CSS using . CSS selector
  • We can have as many classes as we like on the page unlike ID

5. Background styling

Background styling is useful when playing around with styling various elements. This make it easier to identify which element is being styled, and how much space it is taking up on the page

In CSS background properties we can control:

  • background-color
  • background-image
  • background-repeat
  • background-position

You can also use shorthand for the background properties

e.g. div{ background: orange url(/cat.jpg) no-repeat right bottom;}
